Filament — FAQ

How do I send a file from Android to iPhone without an app?

Open filament.autumated.com on both phones. On the same WiFi they detect each other automatically. On different networks, tap create code on one phone, say the code aloud (e.g. clever-lynx-63), and type it on the other. Tap the tile, pick the file, done.

Does Filament upload my files anywhere?

No. The file streams directly between the two browsers over an encrypted WebRTC data channel. When networks force a relay, the relay forwards encrypted packets it cannot read. Nothing is stored, ever.

Is there a size limit?

No fixed limit — there's no storage to fill. Both devices stay online during the transfer; interrupted transfers pause and resume from the same byte.

What do LAN / P2P / RELAY mean?

The badge on each device shows the real route: LAN — straight across your WiFi, the bytes never touch the internet; P2P — direct between devices across the internet; RELAY — via an encrypted relay when both networks block direct connections.

How do the one-time codes work?

A code like quiet-lynx-42 admits exactly one person and burns on first use — overhearing it afterwards gives an attacker nothing. Mint another code to add another person.

Can I self-host it?

Yes — the whole stack is open source: a small Flask signaling server, Redis, and an optional coturn relay. Deploy guide in the repo: github.com/Abdk4Moura/filament.
